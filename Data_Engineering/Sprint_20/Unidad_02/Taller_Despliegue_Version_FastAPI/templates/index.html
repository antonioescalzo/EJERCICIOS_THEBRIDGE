Advertising Model API

This API provides predictions for sales based on advertising budgets.

Available Endpoints:

GET /api/v1/predict

Predicts sales based on advertising budgets.

Parameters:

  • tv - TV advertising budget
  • radio - Radio advertising budget
  • newspaper - Newspaper advertising budget

Example: /api/v1/predict?tv=100&radio=20&newspaper=30

GET /api/v1/retrain

Retrains the model using new data.

Note: Requires updated data in data/Advertising_new.csv
